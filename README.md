# Deutsch

Dieses Projekt erstellt eine persönliche Website mit Seiten in Deutsch, Englisch und Chinesisch. Die Website enthält Abschnitte für Kontaktinformationen, eine Indexseite, eine Liste der Projekte und einen Lebenslauf. Die Verzeichnisstruktur ist so organisiert, dass der englische (`en`) und der chinesische (`cn`) Inhalt getrennt sind. Das Verzeichnis `files` enthält zusätzliche Ressourcen und das Verzeichnis `images` enthält alle auf der Website verwendeten Bilder. Die Datei `styles.css` wird zum Stylen der Website verwendet.

# English

This project establishes a personal website with pages in Germany, English and Chinese. The website contains sections for contact information, an index page, a list of projects, and a resume. The directory structure is organized to separate the English (`en`) and Chinese (`cn`) content. The `files` directory holds additional resources, and the `images` directory contains all the images used on the website. The `styles.css` file is used for styling the website.

# 中文

该项目建立了一个包含德语、英语和中文页面的个人网站。网站包含联系信息、主页、项目列表和简历等部分。目录结构被组织为分别存放英文 (`en`) 和中文 (`cn`) 内容。`files` 目录保存附加资源，`images` 目录包含网站使用的所有图片。`styles.css` 文件用于网站的样式设置。
